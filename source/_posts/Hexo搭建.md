### hexo配置
http://blog.csdn.net/xuezhisdc/article/details/53130383

新建文章
``` bash
hexo new post "article title"
```
生成部署
``` bash
hexo g   # 生成
hexo d   # 部署
// 或直接
hexo d -g #在部署前先生成
```
